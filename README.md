# Image Batch Resizer

A powerful and efficient tool to resize multiple images at once with support for different formats and quality settings.

## ✨ Features

- Batch resize multiple images simultaneously
- Support for common image formats (JPG, PNG, WebP, etc.)
- Customizable resize options (width, height, percentage)
- Preserve aspect ratio option
- High-quality image processing
- Fast and lightweight
- Built with Python + TypeScript/React (Full-stack capable)

## 🛠️ Tech Stack

- **Backend**: Python
- **Frontend**: React + TypeScript (Mockup Sandbox)
- **API**: Node.js / TypeScript
- **Styling**: Modern UI components
- **Build Tools**: Vite, pnpm
